This project was created for the study of data visualization by Joe Pruskowski, a graduate student at Arizona State University.

Term limits is an often-discussed and politically charged topic. The concept is to present the data using multiple visual techniques and determine which methods present viewers with the best insights.

  • The presentations are interactive and provide for viewer exploration.
  • Viewers can set their own term limits and see the impacts.
  • Legislator details are also available.
  • Check out the fun facts for long-time legislators.
